RFLAGS Bit Layout (x86‑64)

Flags & reserved bits
Bits Purpose
0–31 Same as EFLAGS. Includes common status/control flags:
ZF (Zero) CF (Carry) SF (Sign) OF (Overflow) PF (Parity) AF (Aux Carry) DF (Direction) IF (Interrupt)
32–63 Reserved / unused — treated as 0

Note: In 64‑bit mode, user code reads/writes flags via instructions like PUSHFQ/POPFQ and LAHF/SAHF (subset). Reserved bits remain zero.
