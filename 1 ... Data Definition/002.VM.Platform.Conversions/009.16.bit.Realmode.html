Feature Explanation
16-bit registers You mostly use registers like AX, BX, CX, DX, SI, DI, SP, BP. These registers can hold 16 bits (2 bytes) of data.
1MB memory limit The CPU can only directly address up to 1 megabyte (MB) of memory, from address 00000h to FFFFFh. This is also referred to as a 20-bit address space ($2^{20} = 1MB$).
No memory protection Any program running in this mode can write to any part of memory, including memory used by other programs or the operating system itself. There is no safety mechanism to prevent programs from interfering with each other.
No multitasking You typically run one program at a time. That program has complete control over the CPU and all system resources until it finishes or explicitly yields control.
Direct hardware access Programs can read from and write to hardware devices like the keyboard, screen, or disk controller directly, without needing the operating system's permission or mediation.
